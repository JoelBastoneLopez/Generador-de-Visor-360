# Generador de Visor 360

Una herramienta web sencilla y potente para crear visores panorámicos 360 portátiles. Convierte tus imágenes equirectangulares (JPG/PNG) en un único archivo HTML que puedes enviar fácilmente a tus clientes.

## Características

*   **Portátil**: Genera un solo archivo `.html` con la imagen y el visor integrados.
*   **Universal**: Funciona en cualquier navegador moderno (PC, Mac, iOS, Android).
*   **Sin Instalación**: Tu cliente solo necesita hacer doble clic para ver el recorrido.
*   **Personalizable**: Puedes añadir el título del proyecto o nombre del cliente.

## ¿Cómo usarlo?

1.  Abre el archivo `index.html` en tu navegador.
2.  Escribe el nombre del cliente o proyecto.
3.  Arrastra tu imagen 360 al área de carga.
4.  Haz clic en **"Generar Visor Portátil"**.
5.  ¡Listo! Se descargará tu visor listo para compartir.

## Tecnologías

*   HTML5 / CSS3 (Diseño moderno y responsivo)
*   JavaScript (Vanilla)
*   [Pannellum](https://pannellum.org/) (Librería de visualización 360)
